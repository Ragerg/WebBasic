# 자바스크립트로 게임만들기
## Neon City

### 게임하기 : https://ragerg.github.io/WebBasic/game/game.html

## 게임 방법
- spacebar : 점프
- → : 공격
- 왼쪽 위에 펭귄이 있어야 공격 가능 상태
- 점프중일때, 공격 중일때는 공격 불가
- 적과 만나면 게임오버
- 난이도에 따라 적의 속도 변경
- 난이도에 따라 재공격 가능 시간 변경(easy : 3초 / hard : 5초)
